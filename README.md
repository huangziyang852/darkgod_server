# darkgod_server

DarkGod is a DEMO of online ARPG mobile game.

The server of the game is developed using .Net Framework 4.6.1 and deployed on the Windows Server platform. The database uses MySql5.7.The client was developed using Unity3D (2017.4.33f1).

---

ダークガードはオンラインARPGモバイルゲーム DEMOです。

ゲームのサーバーサイドは.Net Framework 4.6.1を使用して開発され、Windows Serverプラットフォームに展開されます、データベースはMySql5.7を使用します。クライアントサイドはUnity3D（2017.4.33f1）を使用して開発されました。
